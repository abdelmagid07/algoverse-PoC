# transformer_lens >= 3.4 is required for the n_ctx override in from_pretrained
# (HookedTransformer.from_pretrained(..., n_ctx=8192)); older versions leak the
# kwarg to HuggingFace and crash. TL 3.4 in turn pins torch>=2.6, transformers
# >=5.4, and a matching torchvision (see below). Colab already ships torch 2.7,
# where torch.compiler.disable(reason=...) works, so the earlier transformers
# upper bound is no longer needed.
transformer_lens>=3.4.0
torch>=2.6
transformers>=5.4.0
# transformer_lens 3.4 imports torchvision; it must match torch 2.7. Do NOT
# uninstall it (older notebooks did, to dodge a torch/torchvision mismatch).
torchvision>=0.22,<0.23
datasets>=2.18.0
scipy>=1.11.0
scikit-learn>=1.3.0
matplotlib>=3.7.0
numpy>=1.24.0
